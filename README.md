# 儿童图书推荐系统 python771

## 项目概述

儿童图书推荐系统是一个基于协同过滤算法的推荐系统，采用前后端分离的技术架构。该系统适用于项目实训、在线课程任务等多种场景，提供了一套完整的源码、数据库及详尽的文档资料。
